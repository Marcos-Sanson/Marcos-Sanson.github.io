Projects

  • Robot Dog Project and Website
    I built a fully-operational robot similar to the Boston Dynamics Spot agile mobile robot. I collaborated with a partner to develop software in Python, which allows the robot to be remotely controlled via Bluetooth. We conducted extensive testing and debugging to ensure the robot performed well in a variety of environments. The robot utilizes a Raspberry Pi, 3D-printed components, servo motors, ultrasonic sensors, and an LCD display. I also published the project documentation and contributed open-source code.
